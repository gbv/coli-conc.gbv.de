Cocoda Mapping Tool

Information Start

coli-conc enables interoperability between Knowledge Organization Systems (KOS) with focus on German library KOS. It provides a technical infrastructure to manage KOS and KOS mappings in order to facilitate a seamless access and use. This service provided by the VZG offers open source web-based applications that include:

  • KOS Registry for an uniform access to KOS and their metadata in JSKOS format,
  • Concordance Registry to collect, store and manage single concept mappings and complete concordances,
  • Mapping Tool Cocoda to support computer-assisted mappings and assess the quality of mappings, and
  • Software for the conversion of KOS and mappings from their proprietary format to JSKOS as well as to import and export both KOS and mapping data.

The current focus is on library classifications Dewey Decimal Classification (DDC), Regensburg Classification (RVK), and Basisklassifikation (BK) to build complete concordances for German libraries. The infrastructure can also be used for other systems such as Wikidata and GND.

Projects

Some of the major projects in which coli-conc is involved.

BARTOC.org

A database of Knowledge Organization Systems and KOS related registries.

coli-ana

A tool for analysis of synthesized DDC numbers.

Digitaler Assistent (DA-3)

Digital Assistent is a web-based tool for computer-assisted subject indexing.

KENOM

An online portal for numismatics.

Hub Concept Mapping

A method for producing multilateral mapping, storing, and using mappings among many KOS.

MAQUIS

MAQUIS is a tool for measurement of quality and analysis of subject indexing.

CCMapper

CCMapper is a mapping application specialized on mapping vocabularies against the Dewey Decimal Classification (DDC).
